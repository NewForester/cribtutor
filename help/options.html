Usage: cribtutor -d <dir> -f <file> -s <prefix> -c <n> -h -t -p -r 

    -d | --directory <dir> - the directory in which look for cribsheets (default .)
    -f | --file <file> - the file containing the list of crib sheets (default cribsheet.txt)
    -s | --skipto <prefix> - start with the crib sheet whose name begins with prefix (default start with the first in the list)
    -c | --choices <n> - the number of terms to blank in each question (default 2)
    -h | --help - enter help mode (sets -d help)
    -t | --test - enter test mode (sets -d test)
    -p | --parser - print crib sheets (no quiz)
    -r | --raw - print parser tree (use with -p)
